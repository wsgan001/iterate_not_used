# LaTeX 怎么读？


LaTeX [ˈleɪtɛk] [ˈleɪtɛx] [ˈlɑːtɛx] [ˈlɑːtɛk] **雷泰克**，**拉泰克** 都可以 。




  * 雷泰克斯和拉泰克斯不正确。而且LaTeX的重音是放在雷或拉上。


  * [ˈlɑːtɛx] 注音符号里的x发的不是克斯的音，而是接近“巴赫”的那个赫。
