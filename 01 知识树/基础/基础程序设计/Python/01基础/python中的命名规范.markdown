## 缘由：


在写python和用一些python的库的时候总是感觉有些库的命名有些奇怪，比如pyQT的就都是大写开头。。就非常想确认下到底python的命名规则是什么样的呢？


## 命名规范：




















 	
  * 类Class用驼峰式表示(HelloWorld)

 	
  * 其他的obj等等，都用‘_’隔开(this_is_object)
