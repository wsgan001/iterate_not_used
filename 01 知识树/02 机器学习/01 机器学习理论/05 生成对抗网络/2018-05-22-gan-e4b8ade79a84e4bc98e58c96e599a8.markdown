# **为什么****GAN中的优化器不常用 ****S****GD ？**





 	
  1. SGD容易震荡，容易使GAN训练不稳定，

 	
  2. GAN 的目的是在高维非凸的参数空间中找到纳什均衡点，GAN 的纳什均衡点是一个鞍点，但是SGD 只会找到局部极小值，因为 SGD 解决的是一个寻找最小值的问题，GAN 是一个博弈问题。
