# Hadoop 是什么？




Hadoop和Spark都是开源的分布式计算框架，都是继承的函数式编程的思想，把一个任务拆分到最小单元，可以并行的执行。

核心的设计就是：MapReduce和HDFS（Hadoop Distributed File System）
